Compare product prices with Torob marketplace and display the lowest price to customers.

== Description ==

Torob Price Compare is a powerful WooCommerce plugin that integrates with Torob marketplace to provide real-time price comparison for your products. Help your customers find the best deals while maintaining transparency about market prices.

= Key Features =

* **Real-time Price Comparison**: Compare your product prices with Torob marketplace
* **Persian Language Support**: Full RTL support and Persian interface
* **Smart Caching**: Efficient caching system to reduce API calls and improve performance
* **Flexible Display Options**: Multiple positions to display price comparison on product pages
* **Admin Dashboard**: Comprehensive admin panel with statistics and reports
* **AJAX Integration**: Smooth user experience with AJAX-powered price searches
* **Bulk Operations**: Manage multiple products at once
* **Export Functionality**: Export price data and search logs
* **Responsive Design**: Works perfectly on all devices
* **SEO Friendly**: Structured data support for better search engine visibility

= How It Works =

1. Install and activate the plugin
2. Configure your Torob API settings
3. Choose where to display price comparisons
4. Customers can see price comparisons automatically or search manually
5. Monitor performance through the admin dashboard

= Display Options =

* After product summary
* After product price
* After add to cart button
* Custom position using shortcode `[torob_price_compare]`

= Admin Features =

* **Dashboard**: Overview of search statistics and cache performance
* **Settings**: Configure API, display options, and caching
* **Reports**: Detailed logs of price searches and comparisons
* **Cache Management**: Monitor and manage cached price data
* **Bulk Actions**: Search prices for multiple products simultaneously

== Installation ==

= Automatic Installation =

1. Log in to your WordPress admin panel
2. Go to Plugins > Add New
3. Search for "Torob Price Compare"
4. Click "Install Now" and then "Activate"

= Manual Installation =

1. Download the plugin zip file
2. Upload it to your `/wp-content/plugins/` directory
3. Extract the files
4. Activate the plugin through the WordPress admin panel

= After Installation =

1. Go to WooCommerce > Torob Price Compare
2. Configure your API settings
3. Set your display preferences
4. Test the connection
5. Start comparing prices!

== Frequently Asked Questions ==

= Do I need a Torob API key? =

Yes, you need to register with Torob and obtain an API key to use this plugin. Contact Torob support for API access.

= Will this plugin slow down my website? =

No, the plugin uses smart caching and AJAX requests to ensure optimal performance. Price data is cached to reduce API calls.

= Can I customize the appearance of price comparisons? =

Yes, you can add custom CSS in the plugin settings to match your theme's design.

= Does it work with all WooCommerce themes? =

The plugin is designed to work with most WooCommerce themes. If you encounter any issues, you can use the shortcode for custom positioning.

= Is it compatible with Persian/RTL websites? =

Yes, the plugin has full RTL support and includes Persian translations.

= Can I export price comparison data? =

Yes, you can export search logs and cache data in both CSV and JSON formats from the admin panel.
